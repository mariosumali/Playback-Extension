# Video Speed Controller

A minimal, draggable, and persistent HTML5 video speed controller for Chrome.

## Features
- **Draggable UI**: Position the overlay anywhere on the video.
- **Persistence**: Remembers speed and position across videos.
- **Customizable**: Remap keys and adjust speed steps via the popup.

## Installation
1. Go to `chrome://extensions` > Enable **Developer mode**.
2. Click **Load unpacked** and select this folder.

## Controls

| Key | Action |
| :--- | :--- |
| **D** | Increase Speed |
| **A** | Decrease Speed |
| **R** | Reset Speed (1.0x) |
| **S** | Toggle Overlay |
| **Z** / **X** | Rewind / Advance |

_Customize these in the extension popup._
